# Comments App

A **React Comments App** where users can post comments with their name and password. Users can **like, delete, and toggle password visibility** for each comment. The app features a modern UI with color-coded user initials.

---

## Features

- Add a comment with **Name, Password, and Comment**  
- **Show/Hide Password** for input form and each comment  
- **Like** comments  
- **Delete** individual comments  
- **Clear All Comments** with a single button  
- Randomized **avatar background colors** for user initials
